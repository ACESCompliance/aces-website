# ACES Compliance Systems — Website

Marketing website for ACES Compliance Systems LLC.

- **Live:** https://acescompliancesystems.netlify.app
- **Stack:** Static HTML/CSS, hosted on Netlify
- **Forms:** Netlify Forms (consultation-request, scorecard-download)
- **Deploy:** Push to `main` — Netlify auto-deploys

## Files

- `index.html` — main landing page (17 sections)
- `scorecard-thanks.html` — thank-you page after scorecard download
- `ACES_OSHA_Warehouse_Compliance_Scorecard.pdf` — lead-magnet PDF
- `logo.png` — brand logo
